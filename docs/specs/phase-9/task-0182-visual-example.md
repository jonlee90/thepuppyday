## Color Scheme (Clean & Elegant Professional)

- **Background**: White card (#FFFFFF) on cream page (#F8EEE5)
- **Icon Background**: Light cream (#EAE0D5)
- **Icon Color**: Charcoal (#434E54)
- **Primary Button**: Charcoal (#434E54) with white text
- **Secondary Buttons**: White with charcoal border
- **Selected Preset**: Charcoal background (#434E54) with white text
- **Unselected Preset**: White background with charcoal text
- **Warning Background**: Orange tint (#FFB347/10)
- **Info Background**: Blue tint (blue-50)
- **Range Slider**: DaisyUI default with charcoal accent

---

## Interactive States

### Preset Buttons
- **Default**: White background, charcoal border, charcoal text
- **Hover**: Light cream background (#EAE0D5)
- **Selected**: Charcoal background (#434E54), white text
- **Active**: Slightly darker charcoal (#363F44)

### Save Button
- **Default (enabled)**: Charcoal background, white text
- **Hover**: Darker charcoal (#363F44)
- **Disabled**: Gray background (#E5E5E5), gray text (#9CA3AF)
- **Loading**: Spinner animation with "Saving..." text

### Range Slider
- **Track**: Light gray
- **Fill**: Charcoal (#434E54)
- **Thumb**: Charcoal circle with shadow

### Number Input
- **Default**: White background, light border
- **Focus**: Charcoal border, subtle shadow
- **Hover**: Slightly darker border

---

## Responsive Behavior

### Desktop (1024px+)
- Full width within container
- Preset buttons in single row
- Range slider at full width

### Tablet (768px - 1023px)
- Slightly reduced padding
- Preset buttons may wrap to two rows
- Range slider maintains full width

### Mobile (< 768px)
- Preset buttons stack vertically or wrap
- Range slider scales down
- Timeline visualization adjusts to fit
- Information badges stack vertically

---

## Accessibility Features

- Keyboard navigation for all interactive elements
- ARIA labels on icons and buttons
- Proper semantic HTML (labels, sections)
- Color contrast meets WCAG AA standards
- Focus indicators on all interactive elements
- Screen reader announcements for state changes

---

## Animation & Transitions

- **Preset button click**: Instant background color change
- **Range slider drag**: Smooth value updates
- **Policy text update**: Instant text replacement
- **Timeline visibility**: Fade in/out when cutoff changes
- **Save button**: Smooth hover transition (200ms)
- **Success message**: Fade in (auto-dismiss after 3 seconds)
- **Warning alert**: Slide down when cutoff = 0
