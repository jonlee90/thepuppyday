## Workflow Examples

### Workflow 1: Setting Up Initial Hours
1. **Open Component**: All days show default (Mon-Sat 9-5, Sun closed)
2. **Expand Monday**: Click "Edit" to open time range editor
3. **Adjust Times**: Change to desired hours (e.g., 8-4)
4. **Copy to Weekdays**: Click "All Weekdays" to apply to Tue-Fri
5. **Configure Saturday**: Open Saturday, set 10-2
6. **Save**: Click "Save Business Hours"
7. **Confirm**: See success message âœ“

### Workflow 2: Adding Lunch Break
1. **Expand Day**: Click "Edit" on Monday
2. **Modify Range 1**: Change end time to 12:00 PM
3. **Add Range**: Click "+ Add Time Range"
4. **Set Range 2**: Start: 1:00 PM, End: 5:00 PM
5. **Validate**: Ensure no overlap error
6. **Copy**: Apply to other weekdays if desired
7. **Save**: Click "Save Business Hours"

### Workflow 3: Fixing Validation Error
1. **See Error**: Red border on Tuesday, error message shown
2. **Expand Day**: Click "Edit" to see details
3. **Identify Issue**: "Close time must be after open time"
4. **Fix**: Adjust end time to be after start time
5. **Verify**: Error disappears
6. **Save**: Now enabled, click to save
